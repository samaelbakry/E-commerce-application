**GoCart** is a modern and fully responsive e-commerce web application built with Next.js and React, designed to deliver a seamless and secure online shopping experience. The platform allows users to browse products by categories and brands, manage their shopping cart and wishlist, securely authenticate (Register, Login, Forget & Reset Password), update their profile information and password, manage addresses, and place orders using cash or online payment methods.

The application is developed using Next.js, React, and TypeScript with a scalable and well-structured architecture. It integrates RESTful APIs for dynamic data handling and uses efficient state management to ensure smooth performance

GoCart aims to simulate a real-world e-commerce platform while applying modern frontend best practices, clean code principles, reusable component structure, and performance optimization techniques to deliver a professional and production-ready application.
